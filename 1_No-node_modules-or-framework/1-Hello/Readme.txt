Simple Hello-World App
  App folder: Hello
  Tutorial from: https://code.visualstudio.com/docs/nodejs/nodejs-tutorial
  1. Make a file Hello/app.js
    1.1 app.js
          var msg = 'Hello World';
          console.log(msg);
  2. Run file in Terminal
    2.1 Hello> node app.js
        Output
          Hello World
  3. Run Code in Node line-by-line console
    3.1 Hello> node
    3.2 Copy Paste code
  4. Run file in VScode
    4.1 Open File 
    4.2 Click Run without Debugging
  5. Run file in Online console https://es6console.com/
    3.1 Copy Paste code
    3.2 Click run
